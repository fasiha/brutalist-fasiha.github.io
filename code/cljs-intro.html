Introduction

You may not know this, but you want to build a webapp.

You need to do something, say something, see something, coodinate something~~~something that would benefit from a web browser.

This tutorial seeks to be an introduction to web programming for non-web programmers venturing into web apps, or web programmers venturing into Clojure and ClojureScript. HTML and CSS basics, as provided by Khan Academy’s Intro to HTML/CSS: Making webpages or Codecademy’s HTML & CSS courses, are pre-requisites.
